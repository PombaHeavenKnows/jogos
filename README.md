# 🎯 Jogo: Adivinhe o Número

Um jogo simples desenvolvido com **HTML, CSS e JavaScript**, onde o jogador deve adivinhar um número entre **1 e 10** em até **3 tentativas**.

---

## 🧠 Como funciona

* O sistema gera um número aleatório
* O jogador tem **3 chances** para acertar
* O jogo dá dicas:

  * 🔺 Número muito baixo
  * 🔻 Número muito alto
* Exibe resultado final:

  * 🎉 Acertou
  * 💀 Perdeu
